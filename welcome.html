onebox-venueconfig-api

API to get all the venue configurations from any external ticketing system.

To do so, we have used a API Blueprint standard, apiary as an online editing and mocking tool, aglio to document it, getsandbox as a sandbox provider, atom as local editor and mockapi as a local sandbox.

Venue Resource Operations  ¶

Retrieve a concrete resource representation of a venue specified by its id. For a concrete venue a specific version of the venue can be retrieved using the parameter version. If you do not know the version you have 2 possibilities:

  • Getting the latest version by not specifying version nor latest_from.

  • Specify the last valid version that you know as latest_from value. This will cause to get the latest version of the venue if there is one. In case that there is no newer version a 404 is returned.

  • Parameters

    • id (text) … The venue identifier
    • version (optional, number) … Return a specific version of the venue. Mutualy exclusive with latest_from
    • latest_from (optional, number) … Given a version it returns the most recent venue’s version grather than the provided version. Mutualy exclusive with version

Retieve a single venue resource
GET/venueconfig-api/1.0/venues

Venue events Resource Operations  ¶

Retrieve a concrete resource representation of a venue’s seating capacity specified by its id. For a concrete Venue seating capacity a specific version of the event can be retrieved using the parameter version. If you do not know the version you have 2 possibilities:

  • Getting the latest version by not specifying version nor latest_from.

  • Specify the last valid version that you know as latest_from value. This will cause to get the latest version of the venue’s seating capacity if there is one. In case that there is no newer version a 404 is returned.

  • Parameters
    • venue_id (required, text) … Identifies a specific venue.
    • id (optional, text) … Filter by a given seating capacity.
    • version (optional, number) … Return a specific version of the event. Mutualy exclusive with latest_from
    • latest_from (optional, number) … Given a version it returns the most recent event’s version grather than the provided version. Mutualy exclusive with version

Retieve a single venue's Venue seating capacity resource
GET/venueconfig-api/1.0/venues/capacities
